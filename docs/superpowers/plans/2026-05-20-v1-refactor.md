> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restructure the legacy desktop tool into a maintainable Python package while preserving its core research workflows and validating the result against the real dataset directory.

**Architecture:** The refactor will introduce a package-based layout under `src/sztcmh_mri_kidney_metrics`, move pure kidney-metric and vMRE/sADC logic into testable domain modules, wrap filesystem and export behavior in services, and keep PyQt widgets as thin adapters over those services.

**Tech Stack:** Python 3.12+, PyQt5, numpy, pandas, nibabel, unittest
